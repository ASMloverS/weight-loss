## 力量训练动作指导(纯自重)

### 基础力量 A(全身) · 35min

| 动作 | 次数 | 要领 |
|---|---|---|
| 深蹲 | 3×15 | 双脚与肩同宽、脚尖微外展;下蹲至大腿水平,膝盖跟脚尖方向、不内扣,慢下 2s 快上 |
| 跪姿俯卧撑 | 3×8–12 | 双膝着地,身体从膝到肩一条线;肘部 45° 外展,下降至胸接近地面 |
| 臀桥 | 3×15 | 仰卧屈膝、脚跟踩地;夹紧臀部顶髋,膝-髋-肩成直线,顶点停 1s,不靠腰代偿 |
| 平板支撑 | 3×30–45s | 前臂撑地、收腹夹臀;身体一条线,不塌腰不撅臀,呼吸均匀 |
| 弓步蹲 | 2×10/腿 | 步幅约一步大;下蹲至前腿大腿水平、后膝接近地面;前膝不超脚尖,躯干直立 |
| 提踵 | 3×20 | 前脚掌踩台阶边缘;慢起至最高点停 1s、慢落;全程控制不弹震 |
| 死虫 | 3×10/侧 | 仰卧屈膝 90° 抬腿,对侧手脚同时缓慢伸直;腰部始终贴地不拱起 |

### 核心 + 恢复 B(精简) · 20min

| 动作 | 次数 | 要领 |
|---|---|---|
| 平板支撑 | 3×30–45s | 前臂撑、身体一条线、收腹夹臀 |
| 侧平板 | 2×20–30s/侧 | 前臂侧撑、髋部上顶不塌;身体在一个平面 |
| 鸟狗式 | 3×10/侧 | 四足跪姿、对侧手脚同时伸直;核心抗旋转、髋部不晃 |
| 臀桥 | 2×15 | 顶髋至直线、停 1s |
| 猫牛式 + 髋屈肌拉伸 | 2×8 + 30s/侧 | 四足跪姿交替弓背/塌腰;后弓步拉伸髋屈肌 |
